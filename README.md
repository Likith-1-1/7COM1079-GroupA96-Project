> Finding statistical correlation between Minutes Played vs Points Scored (NBA 2022–2023)

> Project Overview
This project investigates the relationship between minutes played (MP) and points scored (PTS) using NBA player data from the 2022–2023 season. The analysis was conducted in R and includes data cleaning, visualisation and correlation testing.

> Key Findings
- Pearson correlation: r = 0.8733  
- Spearman correlation: ρ = 0.9273  
Both tests show a strong positive correlation between MP and PTS.

> Files
- Dataset: nba_data_processed.csv  
- Scripts: Cleaning, histograms, scatterplot and correlation analysis
